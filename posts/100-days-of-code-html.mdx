When you're just getting started with web development, the first thing you'll need to learn is <acronym title="Hyper Text Markup Language">HTML</acronym>. HTML will allow you to structure content on a web page. It's relativly easy to learn, but later down the road you'll find there is some nuance in deciding how to use it correctly.

7: Finish your Cat Photo App
Required Practice
Complete lessons 51-68 here: https://www.freecodecamp.org/learn/2022/responsive-web-design/
Copy all of the HTML from the final lesson into a file on your local machine, then view the page in your browser
Bonus Use a new font
Visit Google Fonts: https://fonts.google.com/ and pick one or two fonts and figure out how to embed them in your HTML (hint: you can click "Select this style +" when you find one you like, and it will tell you where to copy+paste the required HTML)

Next, research how you can apply a font-family to an HTML element.
